# Updates (2026-03-03)

## Summary
- Added VRAM-safe micro-batching for training and evaluation to improve stability on RTX 4050-class GPUs.
- Switched to caption-aware text features using MS-COCO annotations, with optional multi-caption averaging.
- Fixed optimization direction so similarity is minimized (not increased).
- Aligned baseline similarity computation with the same captioned text used in evaluation.

## Details
- Training now accumulates gradients across micro-batches, then applies a single update per iteration.
- Text embeddings are built per image using COCO captions when available; multiple captions can be averaged for robustness.
- Evaluation is done in micro-batches from paths to avoid large tensor loads.
- Baseline similarity is computed using the same caption logic as evaluation so fooling rate is meaningful.

## How It Was Improved
- Corrected the optimization direction so the perturbation minimizes CLIP similarity instead of increasing it.
- Increased semantic diversity by averaging multiple MS-COCO captions per image (when available).
- Reduced VRAM pressure with micro-batching so larger effective batches can be used without OOM.
- Made baseline and evaluation consistent, which provides a reliable fooling-rate measurement.
